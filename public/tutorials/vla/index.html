关于本教程

本教程面向CNN 自动驾驶感知算法工程师，以"从零实现"为核心理念，每章包含：

  • 深度理论讲解：不只是"怎么做"，更解释"为什么这样做"
  • 完整可运行代码：每章一套 Python 实现，带详细注释
  • 实战项目：从 Mini-GPT 到端到端 VLA 系统，层层递进
  • 工程实践：TensorRT 部署、分布式训练、性能优化
